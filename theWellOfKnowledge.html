The Different Types of Cybersecurity

There are 5 main different types of cybersecurity, that anyone should be conscious of in this day and age

  1. 1.) Critical Infrastructure Security:

    This type of security consists of the physical information system devices, which virtually every modern society now relies on.

  2. 2.) Application Security:

    Application security is a critical security measure to protect systems. Application security, or appsec for short, uses a combination of software and hardware methods, in an effort to mitigate the risks associated with external threats.

  3. 3.) Network Security:

    A personal favorite of mine, as I am currently working in the networking field. Network security involves the mitigations of threats external to the network itself.

    Personally, I consider this sub-field of cybersecurity extremely important. By protecting the infrastructure of internal networks and inhibiting access, netsec guards what's essentially the first line of defense of a computing device, which is typically the network with which the computer is associated.

  4. 4.) Cloud Security:

    A sub-field of cybersecurity that has been slowly growing relatively recently. Cloud security involves the protection of data in cloud resources. Cloudsec is implemented via software-based security tools and are typically provided by cloud providers.

    As cloud computing is increasing in popularity, cloudsec is also growing.

  5. 5.) Internet of Things (IoT) Security:

    IoT refers to the family of devices know as "Internet of Things." This family consists of digital-based devices that are Internet-capable, and are also appliances. Examples are TVs, refrigerators, wifi routers, printers, cameras, Amazon's Alexa, etc . . .
