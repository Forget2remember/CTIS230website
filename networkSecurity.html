Network Security

Network Security is the process of designing security controls that protect the usability and integrity of computer networks and their associated data and ultimately, the users. We have many distinct methods for creating network security controls.

Essentially, we divide these processes into two classes: Egress and ingress network security. Ingress security includes controls for data that originates outside of the control network. This would be any information that comes from any device outside of your network. Media content, application data, server responses, overhead protocol information, and any other remotely-originated data are all examples of egress data. On the other hand, egress security focuses on traffic that originates internally and is bound for external destinations.

Firewalls are a type of packet-filtering device that are capable of ingress and egress filtering. Firewalls are extremely common all across the Internet. Without these network-based security devices to filter packets based on user-specified rules, the security of any publicly-connected network would be dangerously in peril.
